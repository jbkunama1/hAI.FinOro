hAI.FinOro

AI-powered Trading Agent for eToro — LLM signal generation, live market dashboard & Flask web UI. Built for learning.

⭐ View on GitHub 🚀 Quick Start
What it does
Everything in one dashboard 📊
Monitor markets, get AI-powered trading signals, track your portfolio — all from a self-hosted web UI.
📈
Live Market Prices
Real-time prices for BTC, ETH, Gold, Oil, EUR/USD & GBP/USD via eToro Public API. All in one grid.
🤖
LLM Signal Engine
Sends OHLC candle data to any OpenAI-compatible LLM and gets back BUY / SELL / HOLD with reasoning.
📉
SMA Pre-Filter
SMA20 vs SMA50 trend analysis runs before every LLM call — prevents counter-trend noise.
💼
Portfolio View
See all open real-account positions, direction, P&L percentage. Export to CSV anytime.
🔍
Auto ID Resolver
Automatically finds and saves eToro instrument IDs — no manual lookup needed.
🔐
Security First
Keys never in code. Stage 3 orders locked behind commented placeholder. All inputs validated.
Agent Stages
Three-stage trading pipeline 🚦
The agent moves through stages — from pure observation to live trading. Each stage must be deliberately activated.
STAGE 01
👁 Observe
Fetch prices & candles. Calculate SMA20/SMA50. Log trend. No LLM call.
✓ Active
STAGE 02
🤖 Signals
SMA pre-filter → LLM analysis → BUY/SELL/HOLD with reasoning. No orders placed.
✓ Active
STAGE 03
⚡ Live Orders
Execute real trades via eToro Write API. Requires explicit unlock + write key.
🔒 Locked
Get started
Deploy in 4 steps 🚀
# 1. Clone the repo $ git clone https://github.com/jbkunama1/hAI.FinOro.git && cd hAI.FinOro # 2. Configure keys $ cp config.example.json config.json $ nano config.json # fill in API_KEY, USER_KEY, LLM_BASE_URL # 3. Start with Docker $ docker compose up --build -d # 4. Resolve instrument IDs (first run only) # Open: http://localhost:5000/debug/instruments # Click: 💾 Save IDs to config.json
